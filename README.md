# 使い方
Quadrilingual.pyをダウンロードしてください。<br>
ダウンロードしたファイルを実行してください。<br>
何か日本語で文章を入力してみてください。<br>
すると、エセクァドリンガルの文が出力されます。<br>
これでエセクァドリンガルになれます。<br>
ちなみに言語は、日本語、英語、中国語、韓国語で構成されています。<br>
終了機能はないので、Ctrl-Cをしてください。
